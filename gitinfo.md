# Что такое Git?

# Команды

1. Задать имя и почту

     * git config user.name "name"

     * git config user.email "name@.com"

2. Создать репозиторий

     * git init

3. Работа с репозиторием

    1. Работа с файлами

     * git add

     * git comit

    2. Смотреть состояние

     * git status

     * git log

     * git diff
